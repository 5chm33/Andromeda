# Andromeda v6.21 — Honest Assessment & Roadmap

## 1. The OpenRouter Cost Bug (Fixed)

**What happened:** The `$20.50/hour` cost drain was caused by a stale setting in your `.env.local` file. Specifically, `LLM_MODEL=openrouter` was set on line 47. Because of how the provider logic was structured, this explicitly forced all calls to route through OpenRouter. And because OpenRouter was configured to default to Claude Opus, every background daemon and simple chat was hitting a `$15/M` token model.

**The Fix in v6.21:**
1. I edited your `.env.local` to remove `LLM_MODEL=openrouter` and explicitly set `OPENROUTER_MODEL=google/gemini-2.5-flash` as a safety net.
2. I added a **Cost Guard** to `server/llmProvider.ts`. If it detects `LLM_MODEL=openrouter` *and* a DeepSeek API key is present, it will automatically intercept the routing, redirect to DeepSeek (`~$0.14/M`), and emit a loud warning in the console. This prevents stale config files from ever causing this kind of cost drain again.
